About This Project

The Hand Jumper Panel Database is a fan-made searchable archive of panels from the webtoon Hand Jumper by SLEEPACROSS. This project was created as a portfolio piece and learning exercise in web scraping, data organization, and static site deployment.

Features

  • Searchable Database: Find panels by character, location, tags, or keywords
  • Advanced Filtering: Filter by season, episode range, characters, and more
  • High-Quality Storage: Both original quality and web-optimized images are preserved
  • Manual Tagging: Comprehensive tagging system for detailed categorization
  • Static Deployment: No backend required, fully deployable on GitHub Pages

Technology Stack

  • Backend Scripts (Python):
    • Web scraping with httpx and BeautifulSoup4
    • Panel extraction using OpenCV
    • Image optimization with Pillow
  • Frontend:
    • Vanilla HTML, CSS, and JavaScript
    • Fuse.js for fuzzy search functionality
    • Lazy loading for performance optimization
    • Responsive design for mobile and desktop
  • Data Storage:
    • JSON-based database for static deployment
    • Dual image storage (original + optimized)

How It Works

  1. Scraping: Python scripts download episodes from webtoons.com with rate limiting
  2. Panel Extraction: Computer vision algorithms split episodes into individual panels
  3. Optimization: Images are compressed for web while preserving originals
  4. Tagging: Hybrid approach - automated basic tags + manual detailed tags
  5. Search: Fuse.js enables fast, fuzzy searching through panel metadata
